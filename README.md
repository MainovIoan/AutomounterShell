AutomounterShell
Shell custom care monteaza automat mountpoint-uri la accesarea prin comanda cd.
Functionalitati:
- shell propriu cu prompt
- executie comenzi externe
- comanda cd implementata intern
- automount pe baza unui fisier de configurare
- TTL pentru mountpoint-uri
- demontare automata daca mountpoint-ul nu este utilizat

Rulare
```bash
chmod +x amsh
./amsh

Configurare
Mountpoint-urile sunt definite in fisierul amsh.conf.
